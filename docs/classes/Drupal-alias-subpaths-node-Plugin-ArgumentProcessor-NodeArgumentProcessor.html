ArgumentProcessorBase.php : 116

Processes the provided context argument with allowed argument types.

public process(ContextParam $context_argument, mixed $allowed_argument_types) : bool

Iterates through each allowed argument type and uses the corresponding argument resolver to attempt to resolve the raw value from the context argument. On a successful resolution, the context argument is updated with the processed value and parameter name.

Parameters
$context_argument : ContextParam

The context parameter to process.
