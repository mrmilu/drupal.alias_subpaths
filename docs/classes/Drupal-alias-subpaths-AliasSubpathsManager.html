Resolves the given path to its internal system path and route info.

public resolve(string $path) : array<string|int, mixed>

This method uses the alias manager to resolve the URL, retrieves the route information, processes context parameters, and returns an array containing the requested path, the resolved internal path, route information, and any processed parameters.

Parameters
$path : string

The external URL path.

Return values
array<string|int, mixed> —

An associative array with the following keys:

  • requested_path: The original URL path.
  • path: The resolved internal system path.
  • route: The route information.
  • params: The processed context parameters.
